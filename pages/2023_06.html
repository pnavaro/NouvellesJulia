Packages#

  • AccessibleOptimization.jl : function optimization with arbitrary structs.

  • BaytesOptim.jl : optimization library for Baytes Modules.

  • ConcurrentSim.jl : a discrete event process oriented simulation framework.

  • ContinuousNormalizingFlows.jl : implementations of Infinitesimal Continuous Normalizing Flows Algorithms.

  • CpuMemMonitor.jl : monitoring CPU and MEMORY usage made easy.

  • DataDrivenAcoustics.jl : data driven acoustic propagation models.

  • DynamicUnits.jl : lightweight + fast physical quantities in Julia.

  • Earth.jl: multivariate adaptive regression splines for Julia.

  • EstimatingEquationsRegression.jl : gEneralized Estimating Equations.

  • FastDifferentiation.jl : generating efficient executables to evaluate derivatives of Julia functions.

  • FiniteVolumeMethod1D.jl : implementation of the finite volume method in 1D.

  • FiniteFields.jl : finite fields, elements of finite fields and modular numbers.

  • FieldFlags.jl : a small package for creating bitfield-like structs.

  • HiddenMarkovModels.jl: HMM modeling, simulation, inference and learning.

  • LinearSegmentation.jl : linear segmentation

  • Loraine.jl : general and low-rank semidefinite optimization.

  • LuxAMDGPU.jl : trigger Package for AMDGPU Support in Lux.jl.

  • MotifCluster.jl : motif-based weighted spectral clustering in Julia.

  • MovingBoundaryProblems1D.jl : implementation of the finite volume method for moving boundary problems in 1D.

  • MultivariateChebyshev.jl : interpolation of functions using Chebyshev polynomials

  • NearestCorrelationMatrix.jl : Compute the nearest correlation matrix.

  • ParametricBodies.jl : parametrically defined shapes in WaterLily.jl

  • OperationsResearchModels.jl : a Julia package for operations research subjects.

  • ProductArrays.jl : a lazy array acting like collect(Iterators.product(it...))

  • QuantileNN.jl : nonparametric quantile regression using a nearest neighbor approach.

  • RustFFT.jl: compute FFTs in Julia with the RustFFT crate.

  • SparseArraysCOO.jl : create sparse vectors and matrices conveniently and efficiently.

  • StableSpectralElements.jl : provably stable discontinuous spectral-element methods for conservation laws.

  • SuperLUDIST.jl : distributed Sparse Matrix Factorizations.

  • SupportPoints.jl: points in the plane whose coordinates are independent and follow standard exponential distributions.

  • SyncSort.jl: synchronised Sorting of multiple arrays on different arcitectures.

  • Sisyphus.jl: a high-performance library for gradient based quantum optimal control.
