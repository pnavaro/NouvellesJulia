qui nous présentera [StochasticWeatherGenerators.jl](https://github.com/dmetivie/StochasticWeatherGenerators.jl).

## Packages

- [Ark.jl](https://github.com/ark-ecs/Ark.jl) Archetype-based Entity Component System (ECS).
- [BasicMesh.jl](https://github.com/LukasOstien/BasicMesh.jl) generate meshes for user defined shapes.
- [BayesDensity.jl](https://github.com/oskarhs/BayesDensity.jl) nonparametric univariate Bayesian density estimation.
- [CodeComplexity.jl](https://github.com/charleskawczynski/CodeComplexity.jl) package for estimating code complexity.
- [Collects.jl](https://github.com/JuliaCollections/Collects.jl) collect the elements of a given collection into a collection of the given type.
- [DataLinter.jl](https://github.com/zgornel/DataLinter) Linting tools for ML workflows, data, code.
- [KernelForge.jl](https://github.com/epilliat/KernelForge.jl) GPU primitives.
- [KernelIntrinsics.jl](https://github.com/epilliat/KernelIntrinsics.jl) functions for invoking GPU-specific memory access instructions.
- [FastHankelTransform.jl](https://github.com/pbeckman/FastHankelTransform.jl) nonuniform fast Hankel transform (NUFHT) using local and asymptotic Bessel function expansions.
- [FastTransposeMatrix.jl](https://github.com/eschnett/FastTransposeMatrix.jl) transpose matrices in memory efficiently via SIMD instructions.
- [FlexUnits.jl](https://github.com/Deduction42/FlexUnits.jl) unit package.
- [Heval.jl](https://github.com/taf-society/Heval.jl) AI-powered forecasting agent.
- [MultiFloats.jl](https://github.com/dzhang314/MultiFloats.jl) fast CPU/GPU-accelerated extended-precision arithmetic.
- [NeuroAnalyzer.jl](https://codeberg.org/AdamWysokinski/NeuroAnalyzer.jl) toolbox for analyzing neurophysiological data.
- [PrecisionCarriers.jl](https://github.com/AntonReinhard/PrecisionCarriers.jl) find imprecisions in chains of arithmetic functions.
- [RandLinearAlgebra.jl](https://github.com/numlinalg/RandLinearAlgebra.jl) randomized linear algebra library.
- [RecursiveFactorization.jl](https://github.com/JuliaLinearAlgebra/RecursiveFactorization.jl) recursive matrix factorization algorithms (for small to medium sized matrices).
- [SignedCholesky.jl](https://github.com/sethkasante/SignedCholesky.jl) (Signed) Factorization for symmetric and Hermitian matrices.
- [SimpleNNs.jl](https://github.com/JamieMair/SimpleNNs.jl) A simple neural network library, optimised for small to medium sized, simple neural networks.
- [SymBasis.jl](https://github.com/cevenkadir/SymBasis.jl) generator of basis with symmetries.
- [Tachikoma.jl](https://github.com/kahliburke/Tachikoma.jl) framework for building interactive terminal applications.
