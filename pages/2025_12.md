## Packages

- [BufferIO.jl](https://github.com/BioJulia/BufferIO.jl) alternative IO interface in Julia inspired by Rust.
- [CasualPlots.jl](https://github.com/Eben60/CasualPlots.jl) graphical user interface (GUI) to simplify the creation of plots in Julia. 
- [DocumenterTypst.jl](https://github.com/lucifer1004/DocumenterTypst.jl) A Documenter.jl plugin for generating documentation in Typst/PDF format.
- [FluxOptics.jl](https://github.com/anscoil/FluxOptics.jl) package for inverse problems in optics with differentiable forward models.
- [JSPlots.jl](https://github.com/s-baumann/JSPlots.jl) interactive javascript plots.
- [LinearAlgebraMPI.jl](https://github.com/sloisel/LinearAlgebraMPI.jl) Distributed sparse matrix operations.
- [Lit.jl](https://github.com/nidoro/Lit.jl) A Simple, Data-Centric Web App Framework.
- [PkgTemplatesShikiPlugin.jl](https://github.com/hsugawa8651/PkgTemplatesShikiPlugin.jl) modern Shiki syntax highlighting to Julia package documentation.
- [StateSpaceDynamics.jl](https://github.com/depasquale-lab/StateSpaceDynamics.jl) library for fitting and analyzing state space models. 
- [StructuralVibration.jl](https://github.com/maucejo/StructuralVibration.jl) generate vibration data for mechanical systems.
- [SymbolicIntegration.jl](https://github.com/JuliaSymbolics/SymbolicIntegration.jl) symbolic integration algorithms. Risch method, Rule-based integration (RUBI), and much more.
- [ToonFormat.jl](https://github.com/toon-format/ToonFormat.jl) Token-Oriented Object Notation file format.
- [TypedJSON.jl](https://github.com/gcalderone/TypedJSON.jl) serialization library prioritizing type fidelity, human-readability, and long-term archival.
- [Vecchia.jl](https://github.com/cgeoga/Vecchia.jl) Vecchia approximations for Gaussian log-likelihoods.
