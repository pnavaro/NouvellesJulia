Janvier 2022#

Bonjour à tous, je vous souhaite une très bonne année.

La dernière version de Julia, sortie il y a une dizaine de jours, est la 1.7.1.

Une décision pour le maintien de la journée Julia doit être prise dans les prochains jours, je ne suis pas optimiste…

Voici un résumé de l’actualité Julia que j’ai suivi pour vous le mois dernier.

Billets de blog#

  • 5 Julia Projects for Beginners — Easy Ideas to Get Started Coding in Julia by Logan Kilpatrick

  • 10 things I love about Julia

  • Testing Julia: Fast as Fortran, Versatile as Python : Quelques exemples de codes Fortran, Python et Julia avec mesures de performance.

  • The most underrated feature of the Julia Programming Language, the package manager

  • Why you should learn Julia, as a beginner / first-time programmer

  • Why DataFrame is not type stable and when it matters

Packages#

  • NISTStatisticalReferenceData.jl : Utilitaires pour manipuler les jeux de données de référence NIST.

  • Shrike.jl: Fast approximate nearest neighbor search with random projection trees.

  • DashVtk.jl: ce package permet de visualiser des fichiers au format Vtk dans votre navigateur.

  • Publish.jl: une alternative intéressante à Documenter.jl en version beta. A suivre.

Vidéos#

  • Understanding automatic differentiation (in Julia) : 80 minutes.

Tutoriels et documentation#

  • Learn Julia For Beginners : The Future Programming Language of Data Science and Machine Learning Explained by Logan Kilpatrick

  • Numerical Computing in Julia: je conseille ce court tutoriel pour toutes les personnes voulant tester quelques problèmes courants d’analyse numérique avec Julia. Pour les débutants, il est accompagné d’un autre tutoriel que l’on peut suivre avant : Julia Programming: a Hands-on Tutorial.

Divers#

  • Double pendule : Programme julia de l’animation d’un double pendule.

Les bulletins précédents sont disponibles sur https://pnavaro.github.io/NouvellesJulia

Abonnez-vous à la liste Julia du CNRS
