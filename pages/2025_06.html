Juin 2025#

Bonjour,

Pour commencer, j’aimerais que vous preniez deux minutes pour remplir cette enquête.

https://plmsurvey.math.cnrs.fr/julia2025

Les résultats vont m’aider à organiser les prochains évènements et pourront justifier des demandes de financement. Je compte sur vous. Merci beaucoup.

Ceci est le dernier billet avant la pause estivale pendant laquelle vous pourrez regarder tranquillement les vidéos du prochain JuliaCon du 22 au 25 juillet. La nouvelle version 1.12 devrait sortir en même temps.

Packages#

  • BlockSparseMatrices.jl AbstractMatrix object that allows to construct sparse block matrices.

  • Bcube.jl library providing tools for the spatial discretization of partial differential equations.

  • Exodus.jl interface for accessing the ExodusII data format for large scale finite element simulations.

  • FastMultipole.jl A fast, flexible, multi-kernel, multi-system implementation of the fast multipole method.

  • FixedSizeArrays.jl Fixed-size multidimensional arrays. An Array-like type with less indirection at the cost of resizing capability.

  • Gaston.jl Julia plotting using gnuplot.

  • H2Trees.jl tree data structures and algorithms for efficient computation in the context of H2 methods.

  • JuliaCheck.jl Code checker.

  • LevelSetMethods.jl Library for level set methods in Julia.

  • LinRegOutliers.jl Direct and robust methods for outlier detection in linear regression.

  • LMDiskANN.jl lightweight library for approximate nearest‐neighbor (ANN) indexing on disk.

  • MeshfreePoints.jl package for generating collocation points inside various domains, with support for variable point density. The primary goal here is to use this for solving PDEs with physics-informed machine learning.

  • NumericalDistributions.jl working with user-defined continuous univariate distributions where the PDF is specified numerically.

  • OperationsResearchModels.jl package for operations research problems.

  • UncertaintyQuantification.jl Uncertainty Quantification.

  • Ziggurats.jl Routines for automatically constructing high-performance non-uniform random number generators.
