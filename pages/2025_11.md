## Packages

- [Ariadne.jl](https://github.com/NumericalMathematics/Ariadne.jl) Jacobian-Free-Newton-Krylov solvers and time integration methods.
- [CasualPlots.jl](https://github.com/Eben60/CasualPlots.jl) provide a graphical user interface (GUI) to simplify the creation of plots.
- [CompetingClocks.jl](https://github.com/adolgert/CompetingClocks.jl) Simplest simulation to use non-Exponential transitions.
- [DearDiary.jl](https://github.com/JuliaAI/DearDiary.jl) lightweight machine learning experiment tracking tool.
- [Diplodocus.jl](https://github.com/cneverett/Diplodocus.jl) package for the transport of particle distribution functions through phase space with the inclusion of continous forces and discrete interactions between particle.
- [FluxOptics.jl](https://github.com/anscoil/FluxOptics.jl) inverse problems in optics with differentiable forward models.
- [GeoDynamo.jl](https://github.com/subhk/GeoDynamo.jl) simulating self-sustained planetary dynamos in rotating spherical shells or full balls.
- [HPRLP.jl](https://github.com/ChenKaihuang/HPRLP.jl) GPU Solver for Linear Programming.
- [LibImageQuant.jl](https://github.com/ericphanson/LibImageQuant.jl) smaller PNGs via color quantization.
- [MatrixSign.jl](https://github.com/MurrellGroup/MatrixSign.jl) package for computing the matrix sign, or more generally, the polar factor of a matrix, which is the orthogonal component of the polar decomposition.
- [MultifractalTools.jl](https://github.com/NicolauUP/MultifractalTools.jl) package for performing multifractal analysis on 1D and 2D data.
- [NonlinearOptimizationTestFunctions.jl](https://github.com/UweAlex/NonlinearOptimizationTestFunctions.jl) testing and benchmarking nonlinear optimization algorithms.
- [ParallelKDE.jl](https://github.com/chrissm23/ParallelKDE.jl) Rapid Parallel Kernel Density Estimation.
- [Raycore.jl](https://github.com/JuliaGeometry/Raycore.jl) Performant ray tracing on CPU and GPU.
- [SafePETSc.jl](https://github.com/sloisel/SafePETSc.jl) package that makes distributed PETSc linear algebra feel like native Julia.
- [SeisKit.jl](https://github.com/JuliaEarth/SeisKit.jl) Fast IO operations with SEG-Y files and other basic utilities for working with seismic data.
- [SolarPosition.jl](https://github.com/JuliaAstro/SolarPosition.jl) Solar positioning algorithms.
- [SparseSparse.jl](https://github.com/sloisel/SparseSparse.jl) invert sparse matrices with sparse inverses.
- [SpheroidalWaveFunctions.jl](https://github.com/uvegege/SpheroidalWaveFunctions.jl) computing angular and radial spheroidal wave functions and their associated eigenvalues.
- [Swirl.jl](https://github.com/atantos/Swirl.jl) interactive learning platform for Julia.
- [TravellingSailorProblem.jl](https://github.com/SpeedyWeather/TravellingSailorProblem.jl) maximising a path determined by the wind field.
- [ZeroOrigin.jl](https://github.com/JuliaReliab/ZeroOrigin.jl) A macro for changing the indices of arrays.
- [ZoneProfilers.jl](https://github.com/nhz2/ZoneProfilers.jl) interface for the Tracy instrumentation-based profiler, designed for multithreaded simulations.
