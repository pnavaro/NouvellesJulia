## Packages

- [AbstractOperators.jl](https://github.com/kul-optec/AbstractOperators.jl) Abstract operators for large scale optimization.
- [AcceleratedDCTs.jl](https://github.com/liuyxpp/AcceleratedDCTs.jl) Device agnostic implementation of 1D/2D/3D Discrete Cosine Transforms.
- [Breeze.jl](https://github.com/NumericalEarth/Breeze.jl) Limited area LES-to-mesoscale atmosphere simulations based on Oceananigans.
- [DifferentialRiccatiEquations.jl](https://github.com/mpimd-csc/DifferentialRiccatiEquations.jl) Solvers for autonomous matrix-valued generalized differential Riccati equations (GDREs) with low-rank solutions.
- [DocumenterTypst.jl](https://github.com/lucifer1004/DocumenterTypst.jl) A Documenter.jl plugin for generating documentation in Typst/PDF format.
- [EEGPlot.jl](https://github.com/Marco-Congedo/EEGPlot.jl) An interactive plot for electroencephalographic (EEG) data.
- [ElasticWave2D.jl](https://github.com/Wuheng10086/ElasticWave2D.jl) GPU-accelerated 2D elastic wave simulation.
- [FastTanhSinhQuadrature.jl](https://github.com/svretina/FastTanhSinhQuadrature.jl) Fast and high-precision numerical integration using Tanh-Sinh (Double Exponential) quadrature.
- [FastInterpolations.jl](https://github.com/ProjectTorreyPines/FastInterpolations.jl) A high-performance fast 1D interpolation package.
- [FormulaCompiler.jl](https://github.com/emfeltham/FormulaCompiler.jl) High‑performance compilation of statistical formulas to zero‑allocation evaluators with FD/AD derivative support, serving as a stable computational foundation for post-estimation analysis and simulation with formula-based linear models.
- [Glossaries.jl](https://github.com/JuliaManifolds/Glossaries.jl) Add a Glossary to be used within your Docstrings.
- [HPCSparseArrays.jl](https://github.com/sloisel/HPCSparseArrays.jl) Distributed sparse matrix operations using MPI.
- [HAdaptiveIntegration.jl](https://github.com/zmoitier/HAdaptiveIntegration.jl) Adaptive numerical integration over simplices and orthotopes.
- [HazReg.jl](https://github.com/FJRubio67/HazReg.jl) Parametric hazard-based regression models.
- [Margins.jl](https://github.com/emfeltham/Margins.jl) Methods for marginal effects, adjusted predictions, and contrasts for post-estimation with linear models.
- [MPSKit.jl](https://github.com/QuantumKitHub/MPSKit.jl) Quantum many-body systems simulation using Matrix Product States (MPS).
- [ParaReal.jl](https://github.com/mpimd-csc/ParaReal.jl) A parareal orchestrator.
- [PortfolioOptimisers.jl](https://github.com/dcelisgarza/PortfolioOptimisers.jl) Portfolio Optimisation library.
- [QILaplace.jl](https://github.com/SUTD-MDQS/QILaplace.jl) Tensor-network implementation of QFT & Discrete Laplace transforms.
- [TrimCheck.jl](https://github.com/tz-lom/TrimCheck.jl) Code validation compatibility with Julia's `--trim` compiler option.
