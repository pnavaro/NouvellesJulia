Juillet 2020¶

Billets de blog¶

  • My First Impression of Julia : Un résumé des caractéristiques du langage Julia.

  • Mapping the French mathematical optimization community par Francois Pacaud. Un exemple d’utilisation du package LightGraph.jl.

Vidéo¶

  • Universal Differential Equations for SciML Modeling and Computation Seminar, Chris Rackauckas. A regarder si vous souhaitez suivre l’évolution du package DifferentialEquations.jl qui est devenu SciML. La production de code Julia de Chris Rackauckas et de son équipe est impressionnante.

Packages¶

  • PkgPage.jl. Pour créer une page web avec votre package. Attention, cela ne remplace pas Documenter.jl, il s’agit d’un complément pour améliorer la visibilité de votre travail.

  • Pluto.jl : Lightweight reactive notebooks for Julia. Package permettant de créer des notebooks interactifs.

Tutoriels et documentation¶
