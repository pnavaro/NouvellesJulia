Juillet-Août 2024#

Bonjour,

Voici les nouvelles Julia de l’été. L’évènement marquant est JuliaCon avec l’habituel State Of Julia

Si il n’y avait qu’une vidéo à regarder, ce serait à mon avis celle-ci : New ways to compile

N’oubliez pas les Julia and Optimization Days à Toulouse les 28,29 et 30 octobre 2024.

Bonne rentrée !

Packages#

  • About.jl know more about the structure or nature of objects in Julia.

  • ArnoldiMethod.jl The Arnoldi Method with Krylov-Schur restart.

  • BeeDeeDee.jl Behavior-Driven Development (BDD) for Julia.

  • CentralDiff.jl Higher-order central finite difference scheme.

  • CUTEst.jl repository of constrained and unconstrained nonlinear programming problems for testing and comparing optimization algorithms.

  • FlashRank.jl Rapid Document Ranking, Powered by Lightweight Models.

  • FlexiJoins.jl fresh take on joining tabular or non-tabular datasets.

  • GALAHAD.jl the Julia interface to the GALAHAD library for smooth optimization problems.

  • GalerkinToolkit.jl Finite-element toolbox.

  • Kezdi.jl package that provides a Stata-like interface for data manipulation and analysis.

  • KernelMatrices.jl package for working with hierarchical matrices.

  • MakieExtra.jl more recipes and tools for Makie plots.

  • Metida.jl fitting mixed-effects models with flexible random/repeated covariance structure.

  • MethodInspector.jl package to inspect the names and types of method arguments.

  • NetSurvival.jl A pure-Julia take on standard net survival routines.

  • NovaML.jl machine learning framework.

  • PoissonDiskSampling.jl n-dimensional Poisson disk sampling.

  • Posets.jl Partially ordered sets fully compatible with Graphs.jl.

  • PropertyGraphs.jl package for working with property graphs.

  • QDistributions.jl quantile distributions and associated functions.

  • RingStarProblems.jl solver of Ring Star Problem variants.

  • RuleMiner.jl package for pattern mining.

  • Runic.jl A code formatter with rules set in stone.

  • StateSpaceDynamics.jl probabilistic state space models.

  • ThreadPinning.jl Readily pin Julia threads to CPU-threads.

  • Vizagrams.jl Integrating diagramming and data visualization.

  • WhereIsMyDocstring.jl Helping you find and include docstrings.
