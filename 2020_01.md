# Janvier 2020

Je vous souhaite une très bonne année 2020. Et voici quelques informations
pour bien commencer l’année.

Julia est dans sa version 1.3.1

- Un exemple simple de metaprogrammation et notamment de l’utilisation de la macro [@generated](https://github.com/thautwarm/GeneralizedGenerated.jl)
- [Une série de benchmarks](https://github.com/nassarhuda/JuliaTutorials)  entre Python, MATLAB et Julia pour les solveurs d’ODE 
- Un package Julia [Tar.jl](https://github.com/JuliaLang/Tar.jl) pour manipuler les archives au format TAR.

- Pour finir, je recommande ces supports écrits par Michael F. Herbst: ["Notes for getting to know the Julia programming language in one day.”](https://github.com/mfherbst/course_julia_day).
